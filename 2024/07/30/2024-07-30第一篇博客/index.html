昨天真的因为被虚拟主机气死了。网络一直不稳定Wordpress的后台都进不去。。而且我的博客去年也没怎么更新，感觉买个主机有点浪费，于是就想把博客搬到GitHub吧

1. 前言

1.1 本文讲什么？

这篇文章讲的内容是利用GitHub Pages部署属于自己的免费的hexo博客。

1.2 GitHub Pages是什么？

GitHub Pages是一个静态站点托管服务，它直接从GitHub上的存储库中获取HTML, CSS和JavaScript文件，可选地通过构建过程运行这些文件，并发布网站。

1.3 Hexo 是什么？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用Markdown（或其他标记语言）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

官网：hexo.io

详细部署方式参考：文档 | Hexo

夜梦星尘TECH同步更新地址：使用GitHub部署轻量级博客HEXO | 免费快捷开箱即用 | 保姆级小白向教程
